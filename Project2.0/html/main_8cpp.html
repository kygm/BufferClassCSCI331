Part II: Command-Line Search for Zip Code Records.

This program builds a primary key index from the CSV data file ("zip_codes.csv") and processes command-line flags of the form -Z[ZipCode] to search for specific Zip Code records. For each search, only the targeted record is loaded into memory.
