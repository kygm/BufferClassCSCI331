Writes a blocked sequence set file. 

The file consists of\+:
\begin{DoxyItemize}
\item A file header (packed using \doxylink{class_buffer}{Buffer})
\item A line with the number of blocks
\item For each block\+: a length indicator (the size of the packed block) and the packed block data.
\end{DoxyItemize}
